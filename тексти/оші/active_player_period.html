active_player_period — атрибут Customer.io

Что это: период суток, когда игрок обычно заходит на сайт. Используется маркетингом для отправки рассылок в активное время игрока.

Атрибут

ИмяТипЗначения
active_player_periodstringсм. таблицу ниже или unknown

Периоды (локальное время игрока)

ЗначениеЧасы
late_night00:00–03:59
early_morning04:00–07:59
morning08:00–11:59
afternoon12:00–15:59
evening16:00–19:59
night20:00–23:59
unknownданных мало или нет таймзоны

Алгоритм

  1. Берём последние 30 входов игрока на сайт (поле session.start_time, UTC).
  2. Если входов < 5 или у игрока нет таймзоны → unknown, выходим.
  3. Каждый start_time переводим в локальное время игрока, берём только час (0–23).
  4. Считаем циркулярное среднее часа (см. ниже — обычное среднее не подходит, час — это круг).
  5. По средней часе → бакет из таблицы.
  6. Пишем в Customer.io через API.
Важно: обычное среднее arithmetic не использовать. Игрок входит в 23:00 и в 01:00 → среднее не 12:00, а ~00:00. Только циркулярное.

Формула (циркулярное среднее)

angle_i = hour_i * 15°       // 360 / 24
S = Σ sin(angle_i)
C = Σ cos(angle_i)
mean_hour = (atan2(S, C) в градусах / 15) mod 24

Python

import math

hours = [23, 0, 1, 22, 0]                    # локальные часы входов
ang = [h * math.pi / 12 for h in hours]
S = sum(math.sin(a) for a in ang)
C = sum(math.cos(a) for a in ang)
mean_hour = (math.atan2(S, C) * 12 / math.pi) % 24
# bucket по mean_hour

SQL (PostgreSQL)

SELECT
  user_id,
  MOD(
    DEGREES(ATAN2(
      SUM(SIN(RADIANS(EXTRACT(HOUR FROM local_time) * 15))),
      SUM(COS(RADIANS(EXTRACT(HOUR FROM local_time) * 15)))
    )) / 15 + 24, 24
  ) AS mean_hour
FROM (
  SELECT user_id,
         (start_time AT TIME ZONE 'UTC' AT TIME ZONE user_tz) AS local_time
  FROM sessions
  WHERE start_time > NOW() - INTERVAL '30 days'
) s
GROUP BY user_id
HAVING COUNT(*) >= 5;

Расписание

Батч-джоб раз в сутки в 03:00 UTC. Считает по всем активным игрокам, пушит в Customer.io.

Тест-кейсы

  • Входы каждый день 19–21 → evening.
  • Входы в 23:00 и 01:00 → late_night (НЕ afternoon).
  • 3 входа всего → unknown.
  • Нет таймзоны → unknown.
  • Входы ровно в 12:00 → afternoon.
