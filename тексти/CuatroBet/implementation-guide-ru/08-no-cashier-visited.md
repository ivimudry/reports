# 08 — Касса не посещена — Руководство по внедрению

**Цепочка:** Касса не посещена (зарегистрированные игроки, которые ни разу не открыли страницу депозита)  
**Три пути:** Bounce (быстрый выход) / Long Session (просматривал, не зашёл в кассу) / Free Spins (играл в демо)  
**Ожидаемые коммуникации:** 4 на путь = 12 всего  
**Предварительные условия из мастер-документа:** События `session_start`, `session_end`, `cashier_opened`, `game_launched`; Атрибуты `last_session_duration_seconds`, `has_played_demo`, `deposit_count`; Сегменты `SEG-BOUNCE`, `SEG-LONG-NO-CASHIER`, `SEG-FS-NO-DEP`

---

## ШАГ 0: ПРЕДПОЛЁТНЫЙ ЧЕК-ЛИСТ

- [ ] Событие `session_end` передаёт `session_duration_seconds`
- [ ] Событие `cashier_opened` срабатывает, когда игрок переходит на страницу депозита
- [ ] Атрибут `has_played_demo` отслеживает демо/фриспин-игру
- [ ] Сегмент `SEG-BOUNCE`: сессия <60 секунд + нет взаимодействия с кассой
- [ ] Сегмент `SEG-LONG-NO-CASHIER`: сессия >600 секунд + без кассы
- [ ] Сегмент `SEG-FS-NO-DEP`: играл в демо/FS + нет депозита
- [ ] Ссылки на демо-игры: `{base_url}/game/{game_id}?demo=true`
- [ ] История просмотров сессии доступна для персонализации (опционально, для Путь B Касание 2)

---

## ШАГ 1: СОЗДАТЬ ТРИ JOURNEY

| Название Journey | Триггер входа | Сегмент |
|-----------------|---------------|---------|
| `08-NC-BOUNCE-JOURNEY` | `session_end` WHERE `session_duration < 60` AND `deposit_count = 0` AND нет события `cashier_opened` | `SEG-BOUNCE` |
| `08-NC-LONG-JOURNEY` | `session_end` WHERE `session_duration >= 600` AND `deposit_count = 0` AND нет события `cashier_opened` | `SEG-LONG-NO-CASHIER` |
| `08-NC-FS-JOURNEY` | `free_spins_used` OR `game_launched` WHERE `is_demo = true` AND `deposit_count = 0` | `SEG-FS-NO-DEP` |

**Глобальный выход для всех:** `deposit_success` → выход, вход в Day 1 Retention Layer.

- [ ] Нет пересечения между путями (игрок попадает только в ОДИН путь на основе поведения)
