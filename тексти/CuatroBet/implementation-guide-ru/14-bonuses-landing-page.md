## A. ИСПРАВЛЕНИЕ КРИТИЧЕСКИХ ПРОБЛЕМ (До редизайна)

### Исправление 1: Обратный отсчёт Lucky Wheel

| Проблема | "357D LEFT" — почти год. Нет срочности. Страница выглядит устаревшей. |
|---------|-----|
| Решение | Заменить на обратный отсчёт до **следующего запланированного розыгрыша** |
| Логика | `IF draw_scheduled → show "Próximo sorteo en {hours}h {minutes}m"` <br> `IF no_draw_scheduled → show "Próximo Lucky Wheel: {date}" teaser` |
| **НЕЛЬЗЯ** | Показывать обратный отсчёт > 7 дней. Если таймер > 7д — скрыть карточку. |

### Исправление 2: Приветственный бонус на спорт

| Проблема | Отображается на видном месте. 99.9% — казино-игроки. Впустую потрачено место. |
|---------|-----|
| Решение | Добавить вкладки-фильтры вверху: **Todo / Casino / Deportes** |
| По умолчанию | Выбрана вкладка Casino |
| Карточка спорта | Видна только во вкладке Deportes |
| Исключения | Если `player.game_preference = "Sports"` → по умолчанию вкладка Deportes |

### Исправление 3: Дублирующиеся приветственные карточки

| Проблема | 2 почти идентичные карточки Slots/Live (150% + 60 FS vs 150%). Каннибализация. |
|---------|-----|
| Решение | Объединить в **одну** главную карточку: `120% + 50 FS` (в соответствии с лестницей Chain 05) |
| Показывать | Одну карточку с чётким CTA |
| Если игрок использовал приветственный бонус | Скрыть карточку полностью, заменить предложением пополнения |

### Исправление 4: Нет визуальной иерархии

| Проблема | Lucky Wheel визуально равен приветственным бонусам |
|---------|-----|
| Решение | Главная карточка (большая, вверху) + вспомогательные карточки (меньше, ниже) |
| Макет | 1 главная (на всю ширину) + 2–3 вспомогательные (сетка в половину ширины) |

## B. ПЕРСОНАЛИЗИРОВАННЫЕ МАКЕТЫ ПО ЖИЗНЕННОМУ ЦИКЛУ

### Макет 1: Игрок до FTD

**Сегмент:** `lifecycle_stage = "Registered" OR "Activated"`

| Позиция | Карточка | Контент (ES) | CTA |
|----------|------|-------------|-----|
| **Главная** (на всю ширину) | Welcome Bonus | `Bono de bienvenida: 120% + 50 giros gratis en Gates of Olympus` | `Depositar ahora` → Cashier |
| Вспомогательная 1 | Lucky Wheel | `Próximo sorteo en {countdown}. Depositá para participar.` | `Ver premios` → Wheel page |
| Вспомогательная 2 | Превью топ-игр | 3 избранных слота из топ-10 (изображения + названия) | `Jugar gratis` → Demo mode |
| Вспомогательная 3 | Руководство «Как» | `Cómo depositar en 2 minutos` (пошаговый визуал) | `Depositar` → Cashier |
| **Скрыто** | Предложения пополнения, VIP-контент, Bonus Shop, Квесты | — | — |

---

### Макет 2: День 1 (Post-FTD, первые 72 часа)

**Сегмент:** `lifecycle_stage = "FTD" OR "Day_1_3"`

| Позиция | Карточка | Контент (ES) | CTA |
|----------|------|-------------|-----|
| **Главная** | Предложение второго депозита | `Tu segundo depósito: 75% + 30 giros en {first_session_game}` | `Depositar` → Cashier |
| Вспомогательная 1 | Оставшиеся приветственные FS | `Te quedan {remaining_fs} giros gratis. Usalos ahora.` | `Jugar` → Slot page |
| Вспомогательная 2 | Lucky Wheel | Реальный обратный отсчёт | `Girar` → Wheel |
| Вспомогательная 3 | Рекомендации игр | На основе поведения в первой сессии | `Probar` → Game |

---

### Макет 3: Регулярный игрок

**Сегмент:** `lifecycle_stage = "Regular"`

| Позиция | Карточка | Контент (ES) | CTA |
|----------|------|-------------|-----|
| **Главная** | Еженедельное пополнение (ротация) | `Recarga semanal: {offer_pct}% + {fs_count} giros. Válido hasta domingo.` | `Activar` → Deposit |
| Вспомогательная 1 | Активные квесты | `Misión activa: {quest_name}. Progreso: {progress}%` | `Ver misión` → Quest page |
| Вспомогательная 2 | Lucky Wheel | Реальный обратный отсчёт | `Girar` |
| Вспомогательная 3 | Избранное Bonus Shop | Топ-2 предмета по выкупу | `Ver tienda` → Shop |

---

### Макет 4: Pre-VIP / VIP

**Сегмент:** `lifecycle_stage IN ("Pre-VIP", "VIP_I", "VIP_II", "VIP_III", "VIP_IV")`

| Позиция | Карточка | Контент (ES) | CTA |
|----------|------|-------------|-----|
| **Главная** | Персональное сообщение | `{crm_owner_name}: {personalized_message}. Tu cashback esta semana: {cashback_amount} ARS.` | `Ver mi cuenta VIP` |
| Вспомогательная 1 | Эксклюзивное предложение | VIP-only предложение (ротация еженедельно) | `Activar` |
| Вспомогательная 2 | VIP-события | Предстоящие турниры, эксклюзивные розыгрыши | `Ver eventos` |
| Вспомогательная 3 | Подобранные игры | AI-подбор на основе истории игры | `Jugar` |
| **Скрыто** | Стандартные приветственные/пополнительные предложения | — | — |

## C. СПЕЦИФИКАЦИИ КОМПОНЕНТА КАРТОЧКИ

### Главная карточка
```
Width: 100%
Height: 280px
Background: Gradient or game-specific banner image
Title: Bold, 24px, white
Subtitle: 16px, white 80% opacity
CTA Button: Primary brand color, 14px bold uppercase
Countdown (if applicable): Live timer, 18px monospace
```

### Вспомогательная карточка
```
Width: 48% (2-column grid with 4% gap)
Height: 160px
Background: Dark card, brand accent border
Title: 16px bold
Description: 13px, 2 lines max
CTA: Secondary button or text link
```

## D. ОТСЛЕЖИВАНИЕ АНАЛИТИКИ

| Событие | Когда | Данные |
|---------|-------|--------|
| `card_impression` | Карточка видна в области просмотра | `card_id`, `lifecycle_stage`, `position` |
| `card_click` | Игрок нажимает CTA | `card_id`, `lifecycle_stage`, `cta_text` |
| `card_conversion` | Игрок завершает действие (депозит, игра и т.д.) | `card_id`, `lifecycle_stage`, `conversion_value` |

**Еженедельный отчёт:**
- CTR карточек по стадии жизненного цикла
- Конверсия главной карточки
- Использование вкладок-фильтров (Casino vs Sports)
- Вовлечённость Lucky Wheel (до/после исправления)

## ЧЕК-ЛИСТ ТЕСТИРОВАНИЯ

- [ ] Игрок до FTD видит приветственную главную карточку, а НЕ предложения пополнения
- [ ] Игрок дня 1 видит главную карточку второго депозита с правильной игрой
- [ ] Регулярный игрок видит главную карточку еженедельного пополнения
- [ ] VIP-игрок видит персонализированное сообщение в главной карточке
- [ ] Lucky Wheel показывает реальный обратный отсчёт (не 357 дней)
- [ ] Карточка Lucky Wheel скрывается, если нет розыгрыша в течение 7 дней
- [ ] Карточка спорта видна только во вкладке Deportes
- [ ] Вкладка Casino по умолчанию для всех неспортивных игроков
- [ ] Дублирующиеся приветственные карточки объединены в одну
- [ ] Визуальная иерархия главная/вспомогательные чёткая
- [ ] Карточки отслеживают показы и клики
- [ ] Мобильная адаптивность (карточки стакаются вертикально)
- [ ] Нет сдвига макета при переключении вкладок
