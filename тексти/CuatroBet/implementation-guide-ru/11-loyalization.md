# 11 — Лоялизация — Руководство по внедрению

**Цепочка:** Вовлечение в программу лояльности (Активные vs Пассивные участники)  
**Приоритет:** Фаза 3 (Неделя 7–8)  
**Зависимости:** Система уровней лояльности активна, отслеживание баланса CC

---

## ШАГ 0: ПРЕДПОЛЁТНЫЙ ЧЕК-ЛИСТ

- [ ] Система уровней лояльности настроена (Standard / Silver / Gold / Platinum / VIP)
- [ ] API начисления, списания и баланса CC функционируют
- [ ] Еженедельные метрики прогресса доступны (очки за неделю, % прогресса по уровню)
- [ ] Теги подсегментов игроков доступны: `top_game_type` = Slots / Live / Aviator / Mixed
- [ ] Система миссий/челленджей готова (или Quest-система из Цепочки 10 может быть повторно использована)
- [ ] Механизм истечения CC активен (скользящие 90 дней)
- [ ] Событие milestone уровня срабатывает при пересечении порога уровня

---

## ОПРЕДЕЛЕНИЯ СЕГМЕНТОВ

### Активный участник программы лояльности
```
loyalty_enrolled = true
AND last_session_date ≤ 7 days ago
AND (loyalty_points_earned_30d > 0 OR deposits_30d ≥ 1)
```

### Пассивный участник программы лояльности
```
loyalty_enrolled = true
AND (last_session_date > 7 days ago OR loyalty_points_earned_30d = 0)
AND account_age > 14 days
```

## УСЛОВИЯ ВЫХОДА

| Условие | Действие |
|---------|----------|
| Active → логин в течение 7 дней после коммуникации | Остаётся в потоке Active, сброс таймеров |
| Active → повышение уровня | Срабатывает AL-C4, сброс еженедельного цикла |
| Passive → логинится после C1 или C2 | Перемещается в поток Active |
| Passive → обменивает CC после C3/C4 | Перемещается в поток Active |
| Passive → CC истекают без действия | Удаляется из loyalty-коммуникаций, перемещается в Цепочку 12 (Реактивация) |

---

## ЧЕК-ЛИСТ ТЕСТИРОВАНИЯ

- [ ] Сегмент Active корректно включает только участников лояльности, активных 7 дней
- [ ] Сегмент Passive корректно исключает не-участников программы лояльности
- [ ] Еженедельный email прогресса показывает верный баланс CC, прогресс уровня
- [ ] Адаптация по подсегментам (Slots/Live/Aviator/Mixed) рендерит корректный контент
- [ ] Буст середины недели корректно применяет множитель 2×
- [ ] Челленджи выходного дня ротируются еженедельно (4-недельный цикл)
- [ ] Milestone уровня срабатывает немедленно по событию повышения
- [ ] Таблица преимуществ уровней соответствует конфигурации
- [ ] Предупреждение об истечении CC срабатывает ровно за 30 дней
- [ ] Последний шанс срабатывает ровно за 7 дней
- [ ] Бонус +100 CC начисляется только при логине в течение 48ч
- [ ] Переход Passive → Active работает (игрок перемещён между потоками)
- [ ] Нет дублирования коммуникаций при переходе игрока в середине недели
