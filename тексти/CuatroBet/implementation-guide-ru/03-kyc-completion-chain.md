# 03 — Цепочка завершения KYC — Руководство по внедрению

**Цепочка:** Завершение KYC (Email → Телефон → Документы)  
**Приоритет:** Высокий  
**Расчётное количество коммуникаций:** 9+ касаний в 3 фазах  
**Ключевой результат:** Верификация телефона автоматически подписывает игроков на SMS-канал (лучший канал: 19.68% click rate)  
**Предварительные условия из основного документа:** События `email_verified`, `phone_verified`, `documents_submitted`, `documents_approved`, `documents_rejected`, `deposit_success`; Атрибуты `kyc_level`, `email_verified`, `phone_verified`, `documents_status`, `cumulative_deposits_ars`; Сегменты `SEG-KYC-NONE`, `SEG-KYC-EMAIL-PENDING`, `SEG-KYC-PHONE-PENDING`, `SEG-KYC-DOCS-PENDING`

---

## ШАГ 0: ПРЕДВАРИТЕЛЬНЫЙ ЧЕК-ЛИСТ

- [ ] Событие `email_verified` срабатывает, когда игрок нажимает на ссылку верификации
- [ ] Событие `phone_verified` срабатывает, когда игрок вводит правильный SMS-код
- [ ] Событие `documents_submitted` срабатывает, когда игрок загружает документы KYC
- [ ] События `documents_approved` / `documents_rejected` срабатывают по завершении проверки
- [ ] Атрибут `cumulative_deposits_ars` обновляется в реальном времени
- [ ] Шаблон бонуса `KYC-EMAIL-REWARD`: 2,300 ARS, без вейджера
- [ ] Шаблон бонуса `KYC-PHONE-REWARD`: 2,300 ARS, без вейджера
- [ ] Шаблон бонуса `KYC-DOCS-REWARD-EARLY`: 4,600 ARS, без вейджера
- [ ] Шаблон бонуса `KYC-DOCS-REWARD-HARD`: 9,000 ARS, без вейджера
- [ ] Верификация телефона автоматически активирует SMS opt-in (настроить в параметрах канала)
- [ ] Диплинк на страницу загрузки документов: `{base_url}/profile/verify`
- [ ] SLA проверки документов KYC определён (цель: 24 ч)

## ШАГ 1: СОЗДАНИЕ JOURNEY

**Название кампании:** `03-KYC-JOURNEY`  
**Тип:** Triggered journey (на основе события)

**Повторный вход:** Нет
