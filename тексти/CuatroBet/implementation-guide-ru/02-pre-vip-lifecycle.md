# 02 — Pre-VIP Жизненный цикл — Руководство по внедрению

**Цепочка:** Pre-VIP Жизненный цикл  
**Приоритет:** Высокий  
**Ожидаемые коммуникации:** 6 касаний + точки принятия решений за 30 дней  
**Предварительные требования из основного документа:** Атрибуты `cumulative_deposits_ars`, `days_since_last_activity`, `game_preference`, `most_played_game_name`; Сегменты `SEG-PREVIP`, `SEG-VIP`

---

## ШАГ 0: ПРЕДПУСКОВОЙ ЧЕК-ЛИСТ

- [ ] Атрибут `cumulative_deposits_ars` обновляется в реальном времени при каждом депозите
- [ ] Атрибут `days_since_last_activity` рассчитывается ежедневно
- [ ] Система назначения CRM-владельца готова (именной менеджер, реальный email, reply-to)
- [ ] Создан шаблон бонуса 15% еженедельного кэшбэка (без вейджера, выплата в понедельник, лимит 625 000 ARS)
- [ ] Создан шаблон бонуса для Касания 3: 60% матч до 208 000 ARS + 30 FS, вейджер 20×, действие 72ч
- [ ] Создан шаблон бонуса для Касания 5: 100% матч до 416 000 ARS + 50 FS, вейджер 15×, действие 72ч
- [ ] Модуль прогнозирования оттока активирован (для мониторинга с Дня 21+)
- [ ] Существует Pre-VIP дашборд в CRM для мониторинга командой
- [ ] Данные о любимом слоте игрока (`most_played_game_name`) доступны

---

## ШАГ 1: СОЗДАНИЕ JOURNEY

**Название кампании:** `02-PVIP-JOURNEY`  
**Тип:** Triggered journey (на основе атрибута)  
**Триггер входа:** `cumulative_deposits_ars >= 3130000` AND `days_since_last_activity <= 14`  
**Повторный вход:** Нет

> **Примечание:** Игроки с 3,130,000–19,105,000 ARS получают офферы по множителю High (×2) до достижения 4,180,000 ARS, затем переходят на Pre-VIP+ (×3).  
**Окно Journey:** 30 дней (с возможностью расширения до 60)  
**Условия выхода:**
- Игрок достигает `cumulative_deposits_ars >= 6260000` (VIP I) → выход в онбординг VIP I
- Прошло 60 дней без VIP I → переход в обычный жизненный цикл

## ШАГ 2: ПОСТРОЕНИЕ ПОСЛЕДОВАТЕЛЬНОСТИ КАСАНИЙ

### 2.1 Касание 1 — Активация Pre-VIP (Персональный Email)

| Параметр | Значение |
|----------|----------|
| **ID кампании** | `02-PVIP-T1-EMAIL` |
| **Канал** | Email |
| **Время** | День 1 (при входе в journey) |
| **От** | Именной CRM-владелец (напр., "Matias de CuatroBet") — НЕ общий |
| **Reply-to** | Реальный email CRM-владельца (или контролируемый почтовый ящик) |

**Системные действия (перед отправкой email):**
1. Назначить именного CRM-владельца на запись игрока
2. Отметить игрока как Pre-VIP в сегментации
3. Добавить игрока на Pre-VIP дашборд мониторинга

**Содержание Email:**
- **Тема (ES):** `👑 Acabas de desbloquear estatus Pre-VIP en CuatroBet`
- **Прехедер (ES):** `Soy {crm_owner_name}, tu manager personal`
- **Текст баннера (ES):** `Desbloqueaste Pre-VIP`
- **Описание баннера:** Премиальный баннер в золотых тонах — значок Pre-VIP с иконкой короны, тёмный люксовый фон, золотые эффекты частиц, место для имени игрока
- **Описание тела:** Личное приветствие от именного CRM-владельца. Поздравление со статусом Pre-VIP. Список преимуществ: 15% еженедельный кэшбэк без вейджера, персонализированные рекомендации игр, быстрые выводы, эксклюзивные предложения. Упоминание близости к VIP I. Приветственный подарок: 500 CC + 20 FS на любимом слоте игрока. Разговорный тон, без жёсткого CTA — подпись CRM-владельца с именем, должностью, фото.
- **CTA:** Нет (разговорный тон, без жёсткого CTA)
- **Подпись:** Имя CRM-владельца, должность, фото при наличии

**Конфигурация бонуса:**
- 500 CC (Cuatro Coins) — зачислить немедленно
- 20 FS на любимом слоте игрока (`most_played_game_name`)

**Действие InApp:**
- Показать баннер разблокировки Pre-VIP при следующем входе игрока

**Шаги настройки:**
1. Добавить email-ноду на старте journey
2. Использовать токены персонализации: `{first_name}`, `{crm_owner_name}`, `{player_favorite_slot}`
3. Установить отправителя как CRM-владельца (не общий "CuatroBet")
4. Добавить параллельное системное действие: зачислить 500 CC + 20 FS
5. Добавить параллельное действие: запланировать in-app баннер на следующий вход

## ШАГ 3: ЧЕК-ЛИСТ ТЕСТИРОВАНИЯ

- [ ] Journey срабатывает, когда совокупные депозиты превышают 3 130 000 ARS
- [ ] Journey НЕ срабатывает для игроков, неактивных >14 дней
- [ ] CRM-владелец назначается корректно при входе
- [ ] Email Касания 1 приходит от CRM-владельца (не от общего отправителя)
- [ ] 500 CC + 20 FS зачислены при Касании 1
- [ ] Кэшбэк активируется на День 3 (проверить выплату в понедельник)
- [ ] Рекомендации игр в Касании 3 соответствуют истории игрока
- [ ] Push Касания 4 показывает корректную оставшуюся сумму
- [ ] Решение Дня 14 корректно определяет квалификацию VIP I
- [ ] Бонус Касания 5 (100% матч) доступен только для не-VIP игроков
- [ ] Мониторинг оттока Дня 21 активируется для игроков с депозитами
- [ ] Выход в онбординг VIP I работает в любой точке journey
- [ ] Расширенное 30-дневное окно имеет сниженную частоту
- [ ] Все email от CRM-владельца имеют корректный reply-to
