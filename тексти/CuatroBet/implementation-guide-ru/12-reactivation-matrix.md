# 12 — Матрица реактивации — Руководство по внедрению

**Цепочка:** Реактивация (2 истории вовлечённости × 3 временных сегмента × 4 коммуникации + наложение уровня ценности + специальный случай VNR)  
**Приоритет:** Фаза 2 (Неделя 5–6)  
**Всего базовых коммуникаций:** 24 + 4 (VNR) = 28 коммуникаций  
**Зависимости:** Модель прогнозирования оттока, сегментация по уровням ценности, все каналы активны

---

## ШАГ 0: ПРЕДПОЛЁТНЫЙ ЧЕК-ЛИСТ

- [ ] Модель прогнозирования оттока активна (ранний вход на 3–4 день для высокого риска)
- [ ] Уровни ценности игроков сегментированы: Micro / Low / Mid / High / Pre-VIP+
- [ ] Фильтр поведенческой вовлечённости доступен: High / Low / Zero engagement
- [ ] Все каналы активны: SMS, Email, In-app popup, Push, Feed
- [ ] Шаблоны бонусов реактивации настроены (см. Раздел F)
- [ ] Deep link на страницу депозита работает
- [ ] API для ручного зачисления / подарков доступен для эскалационных предложений

## ОПРЕДЕЛЕНИЯ СЕГМЕНТОВ

### История вовлечённости

**Active History (AH):**
```
deposit_count ≥ 3
AND total_wagered_ars ≥ 6600000 ARS
AND sessions_30d_before_dormancy ≥ 5
```

**Passive History (PH):**
```
deposit_count < 3
OR total_wagered_ars < 6600000 ARS
OR sessions_30d_before_dormancy < 5
```

### Временные сегменты (с момента последнего входа)

| Сегмент | Вход | Описание |
|---------|------|----------|
| 7-Day | `last_login` 7–13 дней назад | Ранний период неактивности |
| 14-Day | `last_login` 14–20 дней назад | Средний период неактивности |
| 21+ Day | `last_login` ≥ 21 день назад | Глубокий период неактивности |

### Наложение уровня ценности (корректирует суммы предложений)

| Уровень ценности | Диапазон депозитов (за всё время) | Множитель предложения |
|-------------------|-----------------------------------|----------------------|
| Micro | < 105,000 ARS | 0.5× |
| Low | 105,000–2,375,000 ARS | 1× (базовый) |
| Mid | 520,001–7,175,000 ARS | 1.5× |
| High | 1,570,000–19,105,000 ARS | 2× |
| Pre-VIP+ | > 4,180,000 ARS | 3× (+ персональное обращение) |

### Наложение поведенческого фильтра

| Фильтр | Определение | Корректировка канала |
|--------|-------------|---------------------|
| High engagement | Открыл ≥3 коммуникации за последние 30 активных дней | Использовать Email + Push |
| Low engagement | Открыл 1–2 коммуникации за последние 30 активных дней | Начинать с SMS |
| Zero engagement | Открыл 0 коммуникаций за последние 30 активных дней | Только SMS, затем звонок для High+ ценности |
