## A. РЕФЕРАЛЬНАЯ МЕХАНИКА

### Как это работает

```
Referrer (existing player)          Referred (new player)
    │                                     │
    ├─ Gets unique link/code              │
    ├─ Shares via WhatsApp/IG/copy        │
    │                                     │
    │                    ┌────────────────┤
    │                    │ Registers with link
    │                    │ Gets enhanced welcome bonus
    │                    │ Makes FTD
    │                    │ Makes STD ←─── REWARD TRIGGER
    │                    └────────────────┘
    │
    ├─ Receives referral reward (cash + CC)
    └─ Progress toward tier milestone
```

**Почему STD (Second Deposit) как триггер?**
- Защита от злоупотреблений (само-реферал, боты)
- Соответствует реальному созданию ценности (игрок вовлечён, а не просто тестирует)
- Снижает количество мошеннических регистраций

## C. ПРАВИЛА ЗАЩИТЫ ОТ ЗЛОУПОТРЕБЛЕНИЙ — НАСТРОЙКА В GR8 TECH

| Правило | Реализация | Действие при нарушении |
|---------|-----------|----------------------|
| Само-реферал | Блокировка при совпадении IP ИЛИ отпечатка устройства ИЛИ платёжного метода | Отклонение реферала, без наград |
| Совпадение платёжного метода | Реферер и приглашённый не могут иметь общий платёжный метод | Отклонение при валидации STD |
| Право на реферал | Необходимо ≥3 депозита за всё время для генерации кода | Скрыть раздел рефералов если не соответствует |
| Месячный лимит | Максимум 10 успешных рефералов на реферера в месяц | Отключить ссылку после 10, показать "Limit reached" |
| KYC барьер | Полная KYC-верификация required для денежных наград >21,000 ARS | Задержать награду, запросить прохождение KYC |
| Проверка скорости | >5 рефералов за 48ч | Пометить для ручной проверки, задержать награды |

### Очередь проверки мошенничества

Когда срабатывает проверка скорости:
1. Задержать все ожидающие реферальные награды
2. Отправить внутреннее уведомление команде по мошенничеству
3. Ручная проверка в течение 24ч
4. Если легитимно → выдать награды
5. Если мошенничество → аннулировать все реферальные награды, пометить аккаунт реферера

## F. ЖИЗНЕННЫЙ ЦИКЛ ПРИГЛАШЁННОГО ИГРОКА

После регистрации приглашённый игрок входит в стандартный жизненный цикл, НО с улучшенными предложениями:

| Этап | Стандарт | Улучшенный (реферал) |
|------|----------|---------------------|
| Приветственный бонус | 120% + 50 FS | **150% + 60 FS** |
| Подарок за регистрацию | Нет | **10,500 ARS без депозита** |
| Day 1 Layer | Стандартная цепочка 01 | Та же цепочка 01 + бейдж |
| Chain 05 лестница | Стандартные уровни | Те же уровни |

**Важно:** После FTD приглашённые игроки следуют ВСЕМ стандартным цепочкам. Улучшение действует только на регистрацию + FTD.
