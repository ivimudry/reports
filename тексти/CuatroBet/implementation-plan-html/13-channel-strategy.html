🔄 Каскад эскалации (Waterfall)

1. NC Feed (In-app)
Всегда первый. Если пользователь онлайн — показать немедленно
2. Web Push
+15 мин. если NC Feed не был прочитан. Требует opt-in
3. Email
+2 часа если всё ещё не прочитано. Smart Send Time если не срочно
4. Popup (On-site)
При следующем входе на сайт. Одноразовый показ
5. SMS / WhatsApp
+24 часа. Только для критических сообщений (оффер истекает) и value_tier ≥ MID
Cost Guard: SMS/WhatsApp отправляется только если value_tier ≥ VT_MID И estimated_value > 10× message_cost.

✅ Контрольный чек-лист

  • NC Feed настроен и работает в реальном времени
  • Web Push opt-in flow реализован
  • Email домен прогрет, SPF/DKIM/DMARC настроены
  • SMS провайдер подключен (CM.com / другой)
  • WhatsApp Business API подключен
  • Часы тишины 23:00–09:00 ART применяются
  • Smart Send Time активирован для Email + Web Push
  • Frequency caps настроены (маркетинг + триггеры отдельно)
  • Cost Guard для SMS/WhatsApp включён (VT_MID+)
  • Waterfall escalation настроен для каждого типа journey
  • Транзакционные сообщения освобождены от лимитов
