# 16 — Referral Program

**Purpose:** Tap existing active base as acquisition channel (does not currently exist)  
**Impact:** Low-effort, medium-impact

---

## Mechanic

| Role | Action |
|------|--------|
| **Referrer** (existing player) | Gets unique referral link/code, shares via WhatsApp, Instagram, copy-paste |
| **Referred** (new player) | Registers using link, gets enhanced welcome bonus |
| **Reward trigger** | When referred player completes **STD** (second deposit), NOT just FTD |
| **Why STD?** | Protects against abuse, aligns with actual value creation |

---

## Offer Structure

### Referrer Rewards (on referred player's STD)
| Reward | Value |
|--------|-------|
| Cash bonus | 2,000 ARS per successful referral |
| CC bonus | 500 CC per referral |
| Tier milestone | 5 referrals → 10,000 ARS + VIP Lucky Wheel |
| Monthly leaderboard | Top 10 referrers → additional 10,000 ARS cash prize |

### Referred Player Rewards
| Reward | Value |
|--------|-------|
| Enhanced welcome bonus | 150% match (instead of 120%) + 60 FS |
| Registration gift | 500 ARS no-deposit bonus |
| Badge | "Invited by [referrer name]" visible in profile |

---

## Anti-Abuse Rules

| Rule | Detail |
|------|--------|
| Self-referral | Blocked (IP, device fingerprint, payment method match) |
| Same payment method | Cannot be used by both referrer and referred |
| Referrer eligibility | Must have ≥3 deposits to generate referral code |
| Monthly cap | Max 10 successful referrals per referrer per month |
| KYC gate | KYC required for referrer rewards above 1,000 ARS cash |
| Velocity check | >5 referrals in 48h → flag for manual review |

## Expected Impact

| Metric | Target |
|--------|--------|
