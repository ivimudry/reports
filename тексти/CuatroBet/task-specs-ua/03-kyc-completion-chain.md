# 03 — Ланцюжок KYC Верифікації

**Пріоритет:** 🟡 Високий  
**Мета:** Підняти завершення KYC з 20% до 35–40%, перетворивши верифікацію на винагороджувану подорож  
**Три фази:** Email → Телефон → Документи  
**Ключовий побічний ефект:** Верифікація телефону автоматично додає гравця в SMS-канал (найефективніший канал)

---

## ФАЗА 1: Верифікація Email

**Вхід:** Реєстрація завершена

### Тач 1 — Запит Верифікації Email
| Поле | Значення |
|------|----------|
| **ID** | KYC-P1-01 |
| **Таймінг** | T+0 (реєстрація) |
| **Канал** | Попап у сесії + Підтверджувальний email з лінком верифікації |
| **Тригер** | Реєстрація завершена |
| **Сегмент** | Усі щойно зареєстровані гравці |
| **Дія** | Запропонувати верифікувати email |
| **Оффер** | 500 ARS бонус при верифікації, без вейджерингу, можна використати одразу |
| **Копі (ES)** | "Verificá tu email y recibí 500 ARS de regalo al instante" |
| **CTA** | Лінк верифікації в email |
| **Вихід** | Верифіковано → нарахувати 500 ARS → Фаза 2; Ні → Тач 2 |

---

### Точка Рішення — T+15 хв
| **Умова** | Email верифіковано? |
|------|----------|
| **Так →** | Нарахувати 500 ARS → перейти до Фази 2 |
| **Ні →** | Тач 2 — SMS-нагадування |

---

### Тач 2 — SMS-Нагадування (email не верифіковано)
| Поле | Значення |
|------|----------|
| **ID** | KYC-P1-02 |
| **Таймінг** | T+15 хв |
| **Канал** | SMS |
| **Тригер** | Email не верифіковано через 15 хв після реєстрації |
| **Сегмент** | Гравці з неверифікованим email |
| **Копі (ES)** | "CuatroBet: Verificá tu email y recibí 500 ARS al instante: [link]" |
| **CTA** | Лінк верифікації |
| **Вихід** | Верифіковано → 500 ARS → Фаза 2; Ні → T+24 год |

---

### Точка Рішення — T+24 год
| **Умова** | Email верифіковано? |
|------|----------|
| **Так →** | Нарахувати 500 ARS → перейти до Фази 2 |
| **Ні →** | Перейти до Фази 2 в будь-якому разі (email можна верифікувати пізніше) |

---

## ФАЗА 2: Верифікація Телефону

**Вхід:** Після верифікації email АБО через T+24 год незалежно від статусу email

### Тач 3 — Запит Верифікації Телефону
| Поле | Значення |
|------|----------|
| **ID** | KYC-P2-01 |
| **Таймінг** | Одразу після завершення Фази 1 |
| **Канал** | Попап у сесії + SMS з кодом верифікації |
| **Тригер** | Фаза 1 завершена (верифіковано або час вийшов) |
| **Сегмент** | Гравці, що входять у Фазу 2 |
| **Дія** | Запропонувати верифікацію телефону |
| **Оффер** | 500 ARS бонус при верифікації телефону, без вейджерингу |
| **Копі (ES)** | "Verificá tu teléfono y recibí 500 ARS más. Total: 1,000 ARS al instante" |
| **CTA** | Ввести код верифікації |
| **Вихід** | Верифіковано → 500 ARS + авто-підписка SMS; Ні → Тач 4 |
| **Примітки** | **Авто-підписка на SMS-канал — найцінніший побічний ефект.** SMS — найкращий канал (19.68% клік-рейт). |

---

### При Верифікації Телефону:
- Нарахувати 500 ARS
- **Автоматично підписати гравця на SMS-канал**

---

### Тач 4 — Нагадування про Телефон (якщо не верифіковано за 48 год)
| Поле | Значення |
|------|----------|
| **ID** | KYC-P2-02 |
| **Таймінг** | T+48 год після початку Фази 2 |
| **Канал** | SMS |
| **Тригер** | Телефон не верифіковано протягом 48 год |
| **Сегмент** | Гравці без верифікації телефону |
| **Копі** | Одне SMS-нагадування |
| **Вихід** | Продовжити без блокування (гравець може вносити депозити) |

---

## ФАЗА 3: Верифікація Документів

**Вхід:** Сумарні депозити досягли 15,000 ARS І документи ще не подано  
**Причина:** Почати на 30% раніше за жорсткий поріг 22,500 ARS, щоб був час завершити до блокування виведень

### Тач 5 — Запит Верифікації Документів
| Поле | Значення |
|------|----------|
| **ID** | KYC-P3-01 |
| **Таймінг** | День 1 (депозити досягли 15,000 ARS) |
| **Канал** | In-app банер + Email |
| **Тригер** | Сумарні депозити ≥ 15,000 ARS, документи не подано |
| **Сегмент** | Гравці на порозі 15K, неверифіковані |
| **Дія** | Запропонувати завантажити документи |
| **Оффер** | 1,000 ARS бонус при верифікації документів, без вейджерингу |
| **Копі (ES)** | "Completá tu verificación ahora y desbloqueá retiros más rápidos plus un bono de 1,000 ARS" |
| **CTA** | "Verificar ahora" → сторінка завантаження документів |
| **Вихід** | Подано → перевірка → Тач 7; Не подано → Тач 6 |
| **Необхідні документи** | Посвідчення особи + підтвердження адреси (стандартний KYC) |

---

### Тач 6 — Email-Інструкція (День 2)
| Поле | Значення |
|------|----------|
| **ID** | KYC-P3-02 |
| **Таймінг** | День 2 (Фази 3) |
| **Канал** | Email |
| **Тригер** | Документи не подано до Дня 2 |
| **Сегмент** | Неверифіковані гравці на порозі 15K |
| **Тема (ES)** | "Tu cuenta está a un paso del acceso total" |
| **Тіло** | Візуальна інструкція: як завантажити документи, очікуваний час перевірки (24 год), переваги повної верифікації |
| **Оффер** | 1,000 ARS бонус досі дійсний |
| **CTA** | "Subir documentos" |

---

### Точка Рішення — День 3
| **Умова** | Документи подано? |
|------|----------|
| **Так (прийнято)** → | Нарахувати 1,000 ARS, святковий попап, ВИХІД до активного lifecycle |
| **Ні →** | Продовжити — чекати жорсткого порогу 22,500 ARS |

---

### Тач 7 — Жорсткий Поріг Досягнуто (22,500 ARS)
| Поле | Значення |
|------|----------|
| **ID** | KYC-P3-03 |
| **Таймінг** | Коли сумарні депозити досягли 22,500 ARS |
| **Канал** | Попап у сесії + Email + SMS |
| **Тригер** | Депозити перетнули 22,500 ARS (жорсткий поріг KYC), досі неверифіковано |
| **Сегмент** | Гравці на обов'язковому порозі |
| **Дія** | Терміновий запит верифікації — виведення заблоковані до підтвердження |
| **Оффер** | 2,000 ARS бонус (подвоєно від офферу Дня 1) як фінальна нагорода |
| **Копі (ES)** | "Verificación obligatoria para retiros. Completá ahora y recibí 2,000 ARS de bono final" |
| **CTA** | "Verificar ahora" → сторінка завантаження документів |

---

### Тач 8 — Верифікацію Прийнято
| Поле | Значення |
|------|----------|
| **ID** | KYC-P3-04 |
| **Таймінг** | При прийнятті документів |
| **Канал** | Попап у сесії + Система |
| **Дія** | Нарахувати бонус + святкування |
| **Оффер** | Нарахувати 2,000 ARS (або 1,000 ARS якщо верифіковано до жорсткого порогу) |
| **Копі (ES)** | "Verificación completa. Retiros rápidos activados. Tenés acceso al canal VIP." |
| **Вихід** | ВИХІД до активного lifecycle на поточній стадії |

---

### Тач 9 — Верифікацію Відхилено
| Поле | Значення |
|------|----------|
| **ID** | KYC-P3-05 |
| **Таймінг** | При відхиленні документів |
| **Канал** | Email + Черга підтримки |
| **Дія** | Направити в чергу підтримки для ручного вирішення |
| **Тіло** | Пояснити причину відхилення, наступні кроки |
| **Оффер** | Бонусний оффер НЕ знімається — залишається при успішній верифікації |

## Модель Витрат
| Параметр | Значення |
|----------|----------|
| Фаза 1 (email) | 500 ARS на верифікованого |
| Фаза 2 (телефон) | 500 ARS на верифікованого |
| Фаза 3 (документи) | 1,000–2,000 ARS на верифікованого |
| Змішана вартість на зареєстрованого | 400–500 ARS |

## Цільові KPI
| Метрика | Базова | Ціль |
|---------|--------|------|
| Завершення KYC | 20% | 35–40% |
| SMS opt-in рейт | — | +25–40% (від верифікації телефону) |
| Тертя при виведенні | Високе | Знижене (вже верифіковані на момент першого виведення) |
| Захист від шахрайства | — | Раніша верифікація документів для Pre-VIP/VIP |
