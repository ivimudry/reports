**Мета:** Активувати модулі Smart Segmentation та Predictive Churn від GR8 Tech

---

## Модель Сегментації на 5-ти Осях

### Вісь 1: Стадія Lifecycle (визначає яка подорож)
| Стадія | Визначення |
|--------|-----------|
| Зареєстрований | Тільки акаунт |
| Активований | Перша дія, без FTD |
| FTD | <24 год |
| День 1–3 | У вікні Day 1 Retention Layer |
| Pre-STD | Відіграв FTD, другого депозиту ще нема |
| Regular | 2+ депозити, активний за останні 14 днів |
| Pre-VIP | Довічні депозити 150K–300K ARS |
| VIP I–IV | За порогами VIP-програми |
| Під загрозою | Активний, але скор predictive churn вище порогу |
| Неактивний | 3–7 / 8–14 / 15–30 / 31–60 / 60+ днів |

### Вісь 2: Рівень Цінності (визначає вибір офферу)
| Рівень | Довічні Депозити |
|--------|-----------------|
| Мікро | <5,000 ARS |
| Низький | 5,000–25,000 ARS |
| Середній | 25,000–75,000 ARS |
| Високий | 75,000–150,000 ARS |
| Pre-VIP+ | >150,000 ARS |

### Вісь 3: Ігрові Вподобання (визначає контент)
| Вподобання | Опис |
|-----------|------|
| Тільки слоти | Більшість гравців |
| Aviator / crash | Гравці crash-ігор |
| Live Casino | Гравці live-столів |
| Мікс | Слоти + ще щось |

### Вісь 4: Ризик Відтоку (прискорювач)
| Ризик | Скор |
|-------|------|
| Здоровий | <30% |
| Спостереження | 30–60% |
| Під загрозою | 60–85% |
| Критичний | >85% |

### Вісь 5: Статус KYC (гейт)
| Статус | Опис |
|--------|------|
| Неверифікований | Без верифікації |
| Частково верифікований | Email + телефон |
| Повністю верифікований | Документи завершені |

## Як Осі Комбінуються

| Вісь | Роль | Приклад |
|------|------|---------|
| **1 — Lifecycle** | Тригерить подорож | Post-FTD → STD lifecycle |
| **2 — Цінність** | Вибирає рівень офферу | Низький → базовий оффер |
| **3 — Ігор. Вподобання** | Вибирає контент/FS | Слоти → Joker's Jewels FS |
| **4 — Ризик Відтоку** | Прискорює таймінг | Спостереження → відправлено раніше за замовчування |
| **5 — KYC** | Гейтить дії | Частково верифікований → SMS доступний |

**Приклад:** Гравець з низькою цінністю, слоти-вподобання, рівень Спостереження, 2 дні post-FTD → входить у Post-FTD lifecycle (Вісь 1), отримує Low-рівень перезавантаження (Вісь 2), з FS на Joker's Jewels (Вісь 3), відправлено раніше (Вісь 4), через SMS (Вісь 5).
