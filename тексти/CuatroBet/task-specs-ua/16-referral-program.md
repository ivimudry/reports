# 16 — Реферальна Програма

**Мета:** Використати існуючу активну базу як канал залучення (зараз не існує)  
**Вплив:** Низькі зусилля, середній вплив

---

## Механіка

| Роль | Дія |
|------|-----|
| **Реферер** (існуючий гравець) | Отримує унікальне реферальне посилання/код, ділиться через WhatsApp, Instagram, copy-paste |
| **Реферований** (новий гравець) | Реєструється за посиланням, отримує покращений велкам бонус |
| **Тригер нагороди** | Коли реферований гравець завершує **STD** (другий депозит), НЕ просто FTD |
| **Чому STD?** | Захищає від зловживань, відповідає реальному створенню цінності |

---

## Структура Офферів

### Нагороди Реферера (на STD реферованого гравця)
| Нагорода | Вартість |
|----------|----------|
| Грошовий бонус | 2,000 ARS за успішне реферування |
| CC бонус | 500 CC за реферування |
| Етап рівня | 5 рефералів → 10,000 ARS + VIP Lucky Wheel |
| Місячний лідерборд | Топ-10 реферерів → додатковий грошовий приз 10,000 ARS |

### Нагороди Реферованого Гравця
| Нагорода | Вартість |
|----------|----------|
| Покращений велкам бонус | 150% матч (замість 120%) + 60 FS |
| Подарунок при реєстрації | 500 ARS бонус без депозиту |
| Бейдж | "Invited by [ім'я реферера]" видимий у профілі |

## Правила Захисту від Зловживань

| Правило | Деталі |
|---------|--------|
| Само-реферування | Заблоковано (IP, відбиток пристрою, збіг платіжного методу) |
| Той самий платіжний метод | Не може використовуватись і реферером, і реферованим |
| Право реферера | Має мати ≥3 депозити для генерації реферального коду |
| Місячний ліміт | Макс 10 успішних рефералів на реферера на місяць |
| KYC гейт | KYC потрібний для нагород реферера понад 1,000 ARS готівкою |
| Перевірка швидкості | >5 рефералів за 48 год → позначити для ручної перевірки |
