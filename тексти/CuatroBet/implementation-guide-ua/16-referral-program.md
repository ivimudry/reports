## A. МЕХАНІКА РЕФЕРАЛЬНОЇ ПРОГРАМИ

### Як це працює

```
Referrer (існуючий гравець)          Referred (новий гравець)
    │                                     │
    ├─ Отримує унікальне посилання/код    │
    ├─ Ділиться через WhatsApp/IG/copy    │
    │                                     │
    │                    ┌────────────────┤
    │                    │ Реєструється за посиланням
    │                    │ Отримує покращений вітальний бонус
    │                    │ Робить FTD
    │                    │ Робить STD ←─── ТРИГЕР НАГОРОДИ
    │                    └────────────────┘
    │
    ├─ Отримує реферальну нагороду (готівка + CC)
    └─ Прогрес до milestone рівня
```

**Чому STD (Second Deposit) як тригер?**
- Захист від зловживань (самореферал, боти)
- Узгоджено з фактичним створенням цінності (гравець залучений, а не просто тестує)
- Зменшує шахрайські реєстрації

## C. ПРАВИЛА ЗАХИСТУ ВІД ЗЛОВЖИВАНЬ — НАЛАШТУВАТИ В GR8 TECH

| Правило | Імплементація | Дія при порушенні |
|------|---------------|-------------------|
| Самореферал | Блокувати якщо IP АБО відбиток пристрою АБО платіжний метод збігаються | Відхилити реферал, без нагород |
| Однаковий платіжний метод | Реферер + запрошений не можуть мати спільний платіжний метод | Відхилити при валідації STD |
| Право на участь рефереру | Мати ≥3 депозити за весь час для генерації коду | Приховати секцію реферальної програми якщо не відповідає |
| Місячний ліміт | Макс. 10 успішних рефералів на рефереру на місяць | Деактивувати посилання після 10, показати "Limit reached" |
| Блокування KYC | KYC повністю верифікований — обов'язково для грошових нагород >1 000 ARS | Утримати нагороду, запропонувати завершити KYC |
| Перевірка швидкості | >5 рефералів за 48 год. | Позначити для ручної перевірки, утримати нагороди |

### Черга фрод-перевірки

Коли спрацьовує перевірка швидкості:
1. Утримати всі очікуючі реферальні нагороди
2. Надіслати внутрішнє сповіщення команді фроду
3. Ручна перевірка протягом 24 год.
4. Якщо легітимно → звільнити нагороди
5. Якщо шахрайство → анулювати всі реферальні нагороди, позначити акаунт рефереру

Після реєстрації запрошеного гравця він входить у стандартний lifecycle, АЛЕ з покращеними пропозиціями:

| Етап | Стандартний | Покращений реферальний |
|-------|----------|------------------|
| Welcome bonus | 120% + 50 FS | **150% + 60 FS** |
| Подарунок за реєстрацію | Немає | **500 ARS без депозиту** |
| Day 1 Layer | Стандартний Chain 01 | Той самий Chain 01 + бейдж |
| Драбина Chain 05 | Стандартні рівні | Ті ж рівні |

**Важливо:** Після FTD запрошені гравці слідують УСІМ стандартним ланцюжкам. Покращення діє тільки на реєстрацію + FTD.

- [ ] Блокування KYC утримує нагороди >1 000 ARS для неверифікованих реферерів
- [ ] Шеринг через WhatsApp відкривається з попередньо заповненим повідомленням
