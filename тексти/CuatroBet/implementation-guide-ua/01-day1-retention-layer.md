# 01 — Day 1 Retention Layer — Інструкція з впровадження

**Ланцюг:** Day 1 Retention Layer  
**Пріоритет:** Найвищий  
**Орієнтовна кількість комунікацій:** 7 (шлях Slots) + 7 (шлях Aviator) = 14 загалом  
**Передумови з основного документа:** Події `deposit_success`, `session_start`, `session_end`, `game_launched`; Атрибути `game_preference`, `deposit_count`, `ftd_date`; Сегменти `SEG-FTD`, `SEG-D1LAYER`, `SEG-GAME-SLOTS`, `SEG-GAME-AVIATOR`

---

## КРОК 0: ПЕРЕДПОЛЬОТНИЙ ЧЕК-ЛИСТ

Перед побудовою будь-яких кампаній перевірте:

- [ ] Подія `deposit_success` спрацьовує в реальному часі (≤5 секунд) з прапорцем `is_ftd`
- [ ] Подія `session_start` / `session_end` спрацьовує коректно
- [ ] Подія `game_launched` спрацьовує з полем `game_type`
- [ ] Атрибут `game_preference` заповнюється (використовується для розгалуження шляхів)
- [ ] Бонусний шаблон `WBL-S-D1` існує (50 FS на Gates of Olympus, 20× відіграш, 24г термін дії)
- [ ] Бонусний шаблон для Aviator: 5 Free Flights, 24г термін дії
- [ ] Deep links налаштовані: `{base_url}/game/gates-of-olympus`, `{base_url}/game/aviator`, `{base_url}/cashier`
- [ ] Система in-app popup активна з можливістю спрацювання менше секунди
- [ ] SMS-шлюз активний, sender ID = "CuatroBet"
- [ ] App Push активовано (опціонально але рекомендовано — втрачається ~30% гілки T+6г без цього)

---

## КРОК 1: СТВОРЕННЯ ПОДОРОЖІ

**Назва кампанії:** `01-D1RET-JOURNEY`  
**Тип:** Тригерна подорож (на основі подій)  
**Тригер входу:** Подія = `deposit_success` WHERE `is_ftd = true`  
**Повторний вхід:** Немає (одноразово для кожного гравця)  
**Вікно подорожі:** 72 години від входу  
**Умови виходу:**
- Гравець досягає `deposit_count >= 2` в будь-який момент → вихід до Post-FTD каденції
- Минає 72 години → вихід до Реактиваційної драбини (якщо не повернувся) або Post-FTD (якщо повернувся)

---

## КРОК 2: ДОДАТИ РОЗГАЛУЖЕННЯ — Slots vs Aviator

**Позиція:** Одразу після тригера входу  
**Тип розгалуження:** Умовний поділ  
**Умова:**
- **Шлях Aviator:** `game_preference = "aviator"` АБО тип гри першої сесії = `crash`
- **Шлях Slots:** Все інше (за замовчуванням)

Очікуваний розподіл: ~95% Slots / ~5% Aviator
