Documentation for generating, regenerating, and translating captions directly in Airtable.

This guide explains how to prepare tables, create an agent, run generation, regeneration, or translation, review costs, and debug errors without leaving Airtable.

What the app does

AI Caption Agents

Додаток створює caption для Airtable records на основі media column і optional Topic / Title column, які вибрані в agent settings. Він може генерувати primary language captions, перегенеровувати їх із correction prompt, перекладати на інші мови, працювати масово або по вибраних рядках, а також записувати детальний history log.

Приложение создает caption для Airtable records на основе media column и optional Topic / Title column, выбранных в agent settings. Оно может генерировать primary language captions, перегенерировать их с correction prompt, переводить на другие языки, работать массово или по выбранным строкам, а также записывать детальный history log.

The app creates captions for Airtable records using the media column and optional Topic / Title column selected in agent settings. It can generate primary-language captions, regenerate them with a correction prompt, translate them into additional languages, run in bulk or selective mode, and store detailed history logs.

Media Analysis

Перший етап generation. App читає Attachments, групує медіа одного row і просить AI описати, що зображено, які об'єкти, настрій, важливі деталі та що не можна вигадувати.

The first generation stage. The app reads Attachments, groups media for each row, and asks AI to describe the content, objects, mood, key details, and what must not be invented.

Caption Generation

Другий етап generation. App бере media analysis, Topic, Brand Voice, System Prompt, Training Data і створює caption у primary language.

Второй этап generation. App берет media analysis, Topic, Brand Voice, System Prompt, Training Data и создает caption на primary language.

The third stage when multiple languages or Translation mode are selected. The app translates source captions in batches per target language while preserving meaning, CTA, and format.

Режим для переписування вже створених captions. Він працює як Generation, але додає correction prompt і поточний caption з клітинки у media analysis, caption generation і translation prompts, якщо вибрано кілька мов.
