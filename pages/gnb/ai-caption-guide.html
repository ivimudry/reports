01 · Overview

Що робить додаток

Что делает приложение

What the app does

AI Caption Agents

Додаток створює caption для Airtable records на основі media column і optional topic column, які вибрані в agent settings. Він може генерувати primary language captions, перегенеровувати їх із correction prompt, перекладати на інші мови, працювати масово або по вибраних рядках, а також записувати детальний history log.

Приложение создает caption для Airtable records на основе media column и optional topic column, выбранных в agent settings. Оно может генерировать primary language captions, перегенерировать их с correction prompt, переводить на другие языки, работать массово или по выбранным строкам, а также записывать детальный history log.

The app creates captions for Airtable records using the media column and optional topic column selected in agent settings. It can generate primary-language captions, regenerate them with a correction prompt, translate them into additional languages, run in bulk or selective mode, and store detailed history logs.

03 · Agent setup

Як створити агента

Как создать агента

How to create an agent

1

Add Agent

Натисни Add Agent, введи назву агента й відкрий settings.

Нажми Add Agent, введи название агента и открой settings.

Click Add Agent, enter an agent name, and open settings.

2

Provider, API Key, Model

Вибери OpenAI, Gemini або Anthropic. API key шифрується в settings table; після збереження його не треба вводити повторно.

Выбери OpenAI, Gemini или Anthropic. API key шифруется в settings table; после сохранения его не нужно вводить повторно.

Choose OpenAI, Gemini, or Anthropic. The API key is encrypted in the settings table and does not need to be re-entered after saving.

3

Tables

Вибери одну або кілька tables, з якими працює agent. App не дозволить додати другу table, якщо назви колонок відрізняються.

Выбери одну или несколько tables, с которыми работает agent. App не позволит добавить вторую table, если названия колонок отличаются.

Choose one or more tables this agent can work with. The app will not allow a second table when column names differ.

4

Source Columns

Вибери Attachments column. Topic column optional: якщо не вибрати, app генерує тільки з media і prompt settings.

Выбери Attachments column. Topic column optional: если не выбрать, app генерирует только из media и prompt settings.

Choose the Attachments column. Topic column is optional: if it is empty, the app generates from media and prompt settings only.

5

Language bindings

Додай мови. Перша мова є Default / Primary; саме вона генерується першою. Кожну language прив'яжи до caption column.

Добавь языки. Первый язык — Default / Primary; он генерируется первым. Каждый language привяжи к caption column.

Add languages. The first language is Default / Primary; it is generated first. Bind each language to its caption column.

6

Channel

Channel можна брати з column або вибрати selectors. Один channel = специфічний caption для нього; кілька channels = універсальний caption.

Channel можно брать из column или выбрать selectors. Один channel = специфичный caption для него; несколько channels = универсальный caption.

Channel can come from a column or selector values. One channel means a channel-specific caption; several channels mean a universal caption.

7

Optional context

За потреби додай Character Limit, Brand Voice, System Prompt, Training Data або Hashtags. Hashtags можна вимкнути, додавати в кінець caption або писати в окрему column.

При необходимости добавь Character Limit, Brand Voice, System Prompt, Training Data или Hashtags. Hashtags можно выключить, добавлять в конец caption или писать в отдельную column.

Optionally add Character Limit, Brand Voice, System Prompt, Training Data, or Hashtags. Hashtags can be disabled, appended to the caption, or written into a separate column.

05 · Functions

Що робить кожна функція і як отримати потрібний результат

Что делает каждая функция и как получить нужный результат

What each function does and how to get the right result

Media Analysis

Перший етап generation. App читає Attachments, групує медіа одного row і просить AI описати, що зображено, які об'єкти, настрій, важливі деталі та що не можна вигадувати.

Первый этап generation. App читает Attachments, группирует медиа одного row и просит AI описать, что изображено, объекты, настроение, важные детали и что нельзя выдумывать.

The first generation stage. The app reads Attachments, groups media for each row, and asks AI to describe the content, objects, mood, key details, and what must not be invented.

Caption Generation

Другий етап generation. App бере media analysis, Topic, Brand Voice, System Prompt, Training Data і створює caption у primary language.

Второй этап generation. App берет media analysis, Topic, Brand Voice, System Prompt, Training Data и создает caption на primary language.

The second generation stage. The app uses media analysis, Topic, Brand Voice, System Prompt, and Training Data to create a caption in the primary language.

Caption Translation

Третій етап, якщо вибрано кілька мов або Translation mode. App бере source captions і перекладає їх batch-запитом по target language, зберігаючи сенс, CTA і формат.

Третий этап, если выбрано несколько языков или Translation mode. App берет source captions и переводит их batch-запросом по target language, сохраняя смысл, CTA и формат.

The third stage when multiple languages or Translation mode are selected. The app translates source captions in batches per target language while preserving meaning, CTA, and format.

Regeneration

Режим для переписування вже створених captions. Він працює як Generation, але додає correction prompt і поточний caption з клітинки у media analysis, caption generation і translation prompts, якщо вибрано кілька мов.

Режим для переписывания уже созданных captions. Он работает как Generation, но добавляет correction prompt и текущий caption из ячейки в media analysis, caption generation и translation prompts, если выбрано несколько языков.

A mode for rewriting existing captions. It behaves like Generation, but adds the correction prompt and the current cell caption to media analysis, caption generation, and translation prompts when multiple languages are selected.

Existing Data Mode

Skip не чіпає вже заповнені клітинки. Regenerate перезаписує їх новим результатом. Для bulk запусків спочатку краще тестувати на Selective.

Skip не трогает уже заполненные ячейки. Regenerate перезаписывает их новым результатом. Для bulk запусков сначала лучше тестировать на Selective.

Skip leaves filled cells untouched. Regenerate overwrites them with a new result. For bulk runs, test with Selective first.
