01 · Overview

Що робить додаток

Что делает приложение

What the app does

AI Caption Agents

Додаток створює caption для Airtable records на основі attachments і topic. Він може генерувати primary language captions, перекладати їх на інші мови, працювати масово або по вибраних рядках, а також записувати детальний history log.

Приложение создает caption для Airtable records на основе attachments и topic. Оно может генерировать primary language captions, переводить их на другие языки, работать массово или по выбранным строкам, а также записывать детальный history log.

The app creates captions for Airtable records using attachments and topic context. It can generate primary-language captions, translate them into additional languages, run in bulk or selective mode, and store detailed history logs.

03 · Agent setup

Як створити агента

Как создать агента

How to create an agent

1

Add Agent

Натисни Add Agent, введи назву агента й відкрий settings.

Нажми Add Agent, введи название агента и открой settings.

Click Add Agent, enter an agent name, and open settings.

2

Provider, API Key, Model

Вибери OpenAI, Gemini або Anthropic. API key шифрується в settings table; після збереження його не треба вводити повторно.

Выбери OpenAI, Gemini или Anthropic. API key шифруется в settings table; после сохранения его не нужно вводить повторно.

Choose OpenAI, Gemini, or Anthropic. The API key is encrypted in the settings table and does not need to be re-entered after saving.

3

Language bindings

Додай мови. Перша мова є Default / Primary; саме вона генерується першою. Додаткові мови перекладаються з primary caption.

Добавь языки. Первый язык — Default / Primary; он генерируется первым. Дополнительные языки переводятся из primary caption.

Add languages. The first language is Default / Primary; it is generated first. Additional languages are translated from the primary caption.

4

Optional context

За потреби додай Character Limit, Brand Voice, System Prompt або Training Data. Це впливає на tone, structure і правила caption.

При необходимости добавь Character Limit, Brand Voice, System Prompt или Training Data. Это влияет на tone, structure и правила caption.

Optionally add Character Limit, Brand Voice, System Prompt, or Training Data. These affect tone, structure, and caption rules.
