1. SMS TRIGGER WEBHOOK

A standalone campaign SMS TRIGGER WEBHOOK with a single "Send and Receive Data" block. Receives a phone number and message text, then sends an SMS via the CM.com API.

Webhook code

{# Phone number sanitization and formatting #}
{% assign p = trigger.phone | remove: '+' | remove: ' ' | remove: '-' | remove: '(' | remove: ')' | remove: '_' | remove: '.' | remove: '/' %}
{% assign plen = p | size %}
{% assign fc = p | slice: 0 %}
{% if fc == '0' and plen == 10 %}
  {% assign p = p | slice: 1, 9 | prepend: '0027' %}
{% else %}
  {% assign p = p | prepend: '00' %}
{% endif %}

{
  "messages": {
    "msg": [{
      "allowedChannels": ["SMS"],
      "from": "PantherBet",
      "to": [{ "number": "{{p}}" }],
      "minimumNumberOfMessageParts": 1,
      "maximumNumberOfMessageParts": 3,
      "body": { "type": "auto", "content": "{{trigger.message}}" },
      "reference": "cio-{{p}}-{{'now' | date: '%Y%m%d%H%M%S'}}"
    }]
  }
}

2. Валидация номеров

Номер телефона из тригера проходит двухэтапную обработку:

Этап 1 — Очистка

Удаляются символы: + пробел - ( ) _ . /

Этап 2 — Форматирование

  • Локальный формат (10 цифр, начинается с 0): 0649095770 → 0027649095770
  • Международный формат (начинается с 27): 27649095770 → 0027649095770

2. Валідація номерів

Номер телефону з тригера проходить двоетапну обробку:

Етап 1 — Очищення

Видаляються символи: + пробіл - ( ) _ . /

Етап 2 — Форматування

  • Локальний формат (10 цифр, починається з 0): 0649095770 → 0027649095770
  • Міжнародний формат (починається з 27): 27649095770 → 0027649095770

2. Phone Number Validation

The phone number from the trigger goes through a two-step process:

Step 1 — Sanitization

Characters removed: + space - ( ) _ . /

Step 2 — Formatting

  • Local format (10 digits, starts with 0): 0649095770 → 0027649095770
  • International format (starts with 27): 27649095770 → 0027649095770
CM.com формат: все номера должны начинаться с 00 + код страны. Пример: 0027649095770
CM.com формат: усі номери повинні починатися з 00 + код країни. Приклад: 0027649095770
CM.com format: all numbers must start with 00 + country code. Example: 0027649095770
