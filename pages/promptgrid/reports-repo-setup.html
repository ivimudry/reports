1. Цель

Создать отдельный GitHub-репозиторий для хранения и хостинга HTML-отчётов, которые генерирует PromptGrid. Это позволяет делиться отчётами с клиентами по прямой ссылке — без необходимости настраивать сервер.

Решение: GitHub Pages — бесплатный статический хостинг, привязанный к git-репозиторию. Каждый push автоматически обновляет сайт.

1. Мета

Створити окремий GitHub-репозиторій для зберігання та хостингу HTML-звітів, які генерує PromptGrid. Це дозволяє ділитися звітами з клієнтами за прямим посиланням — без необхідності налаштовувати сервер.

Рішення: GitHub Pages — безкоштовний статичний хостинг, прив'язаний до git-репозиторію. Кожен push автоматично оновлює сайт.

1. Goal

Create a dedicated GitHub repository for storing and hosting HTML reports generated by PromptGrid. This allows sharing reports with clients via a direct link — no server setup required.

Solution: GitHub Pages — free static hosting tied to a git repository. Every push automatically updates the site.

4. Хостинг и ссылки

Репозиторий подключён к GitHub Pages. После включения в настройках репозитория все HTML-файлы становятся доступными по прямым ссылкам.

Формула ссылки

Базовый URL: https://ivimudry.github.io/reports/

Для файла в pages/my-report.html ссылка будет:

https://ivimudry.github.io/reports/pages/my-report.html

Рабочий процесс

  • Скопировать шаблон из templates/ в pages/
  • Заменить контент на актуальные данные
  • git add → git commit → git push
  • Ссылка доступна через ~1 минуту

4. Хостинг і посилання

Репозиторій підключено до GitHub Pages. Після ввімкнення в налаштуваннях репозиторію всі HTML-файли стають доступними за прямими посиланнями.

Формула посилання

Базовий URL: https://ivimudry.github.io/reports/

Для файлу в pages/my-report.html посилання буде:

https://ivimudry.github.io/reports/pages/my-report.html

Робочий процес

  • Скопіювати шаблон з templates/ в pages/
  • Замінити контент на актуальні дані
  • git add → git commit → git push
  • Посилання доступне через ~1 хвилину

4. Hosting & Links

The repository is connected to GitHub Pages. Once enabled in repo settings, all HTML files become accessible via direct links.

Link Formula

Base URL: https://ivimudry.github.io/reports/

For a file at pages/my-report.html the link will be:

https://ivimudry.github.io/reports/pages/my-report.html

Workflow

  • Copy a template from templates/ to pages/
  • Replace content with actual data
  • git add → git commit → git push
  • Link is live within ~1 minute
