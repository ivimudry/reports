2. URL Pattern

All href fields follow this unified URL structure:

https://heyhocasino.com/{locale}?token={{customer.auth_token}}
  [&promo-code={CODE}] ← only for emails with a promo code
  &utm_campaign={campaign}
  [&utm_bonus_type=live|slots] ← Nutrition & DEP Retention only
  &utm_source=customerio
  &utm_medium=email
  &utm_language={locale}
  &utm_email={N}
  #modal=Wallet
Note: pirat_girl_href and button_href_3 (Unsuccessful Deposit only) link to /page/contact-us without #modal=Wallet.

Locale mapping: Default → en, de-DE → de, fr-FR → fr, nl-NL → nl

Full URL examples

Deposit (with promo-code):
https://heyhocasino.com/en?token={{customer.auth_token}}&promo-code=XXXXX&utm_campaign=nutrition1&utm_bonus_type=slots&utm_source=customerio&utm_medium=email&utm_language=en&utm_email=1#modal=Wallet
Contact Us (no promo, no #modal):
https://heyhocasino.com/en/page/contact-us?token={{customer.auth_token}}&utm_campaign=faildep&utm_source=customerio&utm_medium=email&utm_language=en&utm_email=3
