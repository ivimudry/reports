# 07 — Кластер без бонуса — Руководство по внедрению

**Цепочка:** Кластер без бонуса (игроки, которые не воспользовались приветственным бонусом)  
**Две ветки:** Missed (не увидели) vs None Interested (сознательно отказались)  
**Ориентировочное количество коммуникаций:** 4 на ветку = 8 всего  
**Необходимые условия из Master Doc:** Атрибуты `welcome_bonus_claimed`, `welcome_bonus_status`; Сегменты `SEG-BONUS-MISSED`, `SEG-BONUS-DECLINED`

---

## ШАГ 0: ПРЕДВАРИТЕЛЬНЫЙ ЧЕК-ЛИСТ

- [ ] Атрибут `welcome_bonus_claimed` отслеживает, воспользовался ли игрок приветственным бонусом
- [ ] Атрибут `welcome_bonus_status` различает: `available` / `claimed` / `declined` / `expired`
- [ ] Реализована логика определения «бонус пропущен» (игрок прошёл через бонусный флоу, не активировав его)
- [ ] Ссылки на демо/бесплатную игру работают: `{base_url}/game/{game_id}?demo=true`
- [ ] Список 5 лучших игр доступен для персонализации email
- [ ] Шаблоны бонусов 200 CC / 300 CC созданы (для усиления в Comm 4)

---

## ШАГ 1: СОЗДАНИЕ ДВУХ JOURNEY

| Название Journey | Триггер входа | Сегмент |
|-----------------|---------------|---------|
| `07-NB-MISS-JOURNEY` | `deposit_count = 0` AND `welcome_bonus_claimed = false` AND `hours_since_registration >= 1` AND нет взаимодействия с бонусом | `SEG-BONUS-MISSED` |
| `07-NB-NONE-JOURNEY` | `welcome_bonus_status = "declined"` OR бонусный флоу пропущен | `SEG-BONUS-DECLINED` |

**Глобальный выход для обоих:** `deposit_success` → выход, переход в Day 1 Retention Layer.

## ШАГ 4: ЧЕК-ЛИСТ ТЕСТИРОВАНИЯ

- [ ] Ветка A корректно определяет игроков, которые ПРОПУСТИЛИ бонус (нет взаимодействия, не отклоняли)
- [ ] Ветка B корректно определяет игроков, которые ОТКЛОНИЛИ бонус (явное действие или обход)
- [ ] Ветка A: все 4 коммуникации прямо упоминают приветственный бонус
- [ ] Ветка B: Comm 1 и Comm 3 НЕ содержат упоминаний бонуса (только знакомство с продуктом)
- [ ] Ветка B: Comm 2 и Comm 4 содержат только МЯГКОЕ упоминание бонуса в футере
- [ ] Демо-ссылки работают корректно: `?demo=true` открывает игру в бесплатном режиме
- [ ] Глобальный выход срабатывает мгновенно при `deposit_success`
- [ ] Бонус 200 CC в Ветке A Comm 4 зачисляется корректно
- [ ] Ограничения частоты: максимум 2 сообщения/день (сегмент pre-FTD)
- [ ] UTM-параметры на всех ссылках
